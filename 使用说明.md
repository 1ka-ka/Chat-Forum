# ChatForum 快速使用指南

## 🚀 第一次使用

### 步骤 1: 准备
1. 确保已安装 Visual Studio 2022 (包含 C++ 组件)
2. 确保已安装 Node.js (16.x 或更高)
3. 确保已安装 MySQL 8.0 或更高

### 步骤 2: 运行安装向导
双击运行 **`setup.bat`**

这个脚本会:
- 检查并安装 vcpkg
- 下载 Drogon 和依赖包 (可能需要30分钟以上)
- 提示你初始化数据库
- 编译后端代码
- 安装前端依赖

### 步骤 3: 初始化数据库
双击运行 **`初始化数据库.bat`**

或者手动执行:
```bash
mysql -u root -p < sql/schema.sql
mysql -u root -p chatforum < sql/init_data.sql
```

### 步骤 4: 启动!
双击 **`快速启动.bat`** 即可启动服务!

## 📖 日常使用

### 启动服务
双击 **`快速启动.bat`**

### 停止服务
直接关闭那两个命令行窗口即可

## 🔧 故障排除

### MySQL 连接失败
编辑 `config/config.json` 修改数据库密码

### CMake 找不到
确保 Visual Studio 2022 已安装 C++ 开发工具

### npm 安装慢
使用国内镜像:
```
npm config set registry https://registry.npmmirror.com
```

## 📂 项目说明

- 后端代码: `backend/`
- 数据库脚本: `sql/`
